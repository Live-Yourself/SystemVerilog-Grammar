# 第5章 SystemVerilog类基础

## 知识点1: 类的定义与实例化

### 类的基本概念

| 特性 | 说明 |
|------|------|
| **定义关键字** | `class` / `endclass` |
| **用途** | 封装数据和方法,创建对象 |
| **存储位置** | 动态内存(堆区) |
| **生命周期** | 从创建到被垃圾回收 |

### 类的定义语法

```systemverilog
// 基本语法
class ClassName;
  // 成员变量(属性)
  data_type variable_name;
  
  // 成员方法(函数/任务)
  function return_type method_name();
    // 方法体
  endfunction
endclass
```

### 对象的声明与创建

SystemVerilog中创建对象需要**两个步骤**:

1. **声明句柄** (Handle Declaration) - 声明一个对象引用
2. **构造对象** (Object Construction) - 调用new()分配内存

```systemverilog
// 步骤1: 声明句柄
ClassName obj_handle;  // 此时为null,未指向任何对象

// 步骤2: 构造对象
obj_handle = new();    // 分配内存,调用构造函数

// 或合并为一步
ClassName obj_handle = new();
```

### 关键要点

- **类是模板**,对象是类的实例
- **句柄是指针**,存储对象的引用地址
- **声明不分配内存**,只有new()才分配
- **多个句柄可指向同一对象**
- **对象必须显式创建**,不会自动初始化

### 典型应用

- **事务(Transaction)**: 验证中的数据包建模
- **驱动器(Driver)**: 发送激励
- **监视器(Monitor)**: 采集信号
- **记分板(Scoreboard)**: 数据比对
- **测试用例(TestCase)**: 测试场景封装
