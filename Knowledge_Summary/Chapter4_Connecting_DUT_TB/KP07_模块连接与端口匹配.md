# 知识点7: 模块连接与端口匹配

| 特性 | 说明 |
|------|------|
| **书中小节** | 4.5 |
| **核心概念** | 端口连接方式、隐式/显式连接、端口匹配规则 |
| **主要作用** | 正确连接模块与接口，避免端口连接错误 |

## 7.1 端口连接方式概述

SystemVerilog提供了多种端口连接方式：

```
┌─────────────────────────────────────────────────────────────┐
│                    端口连接方式对比                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   方式1: 按名称连接（推荐）                                   │
│   ┌─────────────────────────────────────────┐              │
│   │  dut u_dut(                             │              │
│   │    .clk    (clk),      // 端口名(信号名)  │              │
│   │    .rst_n  (rst_n),                     │              │
│   │    .data   (data)                       │              │
│   │  );                                      │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
│   方式2: 按位置连接                                          │
│   ┌─────────────────────────────────────────┐              │
│   │  dut u_dut(clk, rst_n, data);           │              │
│   │           ↑ 顺序必须严格对应端口列表      │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
│   方式3: 接口连接                                            │
│   ┌─────────────────────────────────────────┐              │
│   │  dut u_dut(bus_if.DUT bus);             │              │
│   │           ↑ 使用接口modport              │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 7.2 按名称连接（推荐）

```systemverilog
// 推荐方式：按名称连接
module top;
  logic       clk;
  logic       rst_n;
  logic [7:0] data_in;
  logic [7:0] data_out;
  
  // 按名称连接 - 清晰、易维护、顺序无关
  simple_dut u_dut (
    .clk      (clk),       // 端口名(连接信号)
    .rst_n    (rst_n),
    .data_in  (data_in),
    .data_out (data_out)
  );
endmodule
```

**优点：**

| 优点 | 说明 |
|------|------|
| **清晰易读** | 一目了然哪个端口连接哪个信号 |
| **顺序无关** | 端口顺序可以任意排列 |
| **易于维护** | 修改端口顺序不影响连接 |
| **减少错误** | 编译器检查端口名称是否匹配 |

## 7.3 按位置连接（传统方式）

```systemverilog
// 传统方式：按位置连接
module top;
  logic       clk;
  logic       rst_n;
  logic [7:0] data_in;
  logic [7:0] data_out;
  
  // 按位置连接 - 顺序必须严格匹配
  simple_dut u_dut (clk, rst_n, data_in, data_out);
endmodule
```

**缺点：**

| 缺点 | 说明 |
|------|------|
| **易出错** | 顺序错误难以发现 |
| **可读性差** | 无法直观看出连接关系 |
| **维护困难** | 修改端口顺序需要修改所有实例 |
| **不推荐** | 仅用于端口较少的简单模块 |

## 7.4 接口连接方式

```systemverilog
// 方式1: 接口作为端口
interface bus_if;
  logic       clk;
  logic       rst_n;
  logic [7:0] data;
  logic       valid;
  
  modport DUT (input clk, rst_n, data, valid);
  modport TB  (output clk, rst_n, data, valid);
endinterface

module dut(bus_if.DUT bus);  // 使用modport
  always_ff @(posedge bus.clk) begin
    // 使用 bus.data, bus.valid
  end
endmodule

module top;
  bus_if bus();
  
  dut u_dut(bus);  // 直接传递接口实例
endmodule
```

## 7.5 隐式端口连接 `.*`

SystemVerilog支持隐式端口连接，简化连接代码：

```systemverilog
module top;
  logic       clk;
  logic       rst_n;
  logic [7:0] data_in;
  logic [7:0] data_out;
  
  // 隐式连接：自动连接同名的端口和信号
  simple_dut u_dut (.*);
  
  // 等价于:
  // simple_dut u_dut (
  //   .clk      (clk),
  //   .rst_n    (rst_n),
  //   .data_in  (data_in),
  //   .data_out (data_out)
  // );
endmodule
```

**隐式连接规则：**

| 规则 | 说明 |
|------|------|
| **名称匹配** | 端口名与信号名必须相同 |
| **类型兼容** | 类型必须兼容 |
| **可以混合** | 可与显式连接混合使用 |

```systemverilog
// 混合使用：部分隐式，部分显式
simple_dut u_dut (
  .*,              // 隐式连接同名信号
  .data_in (my_data)  // 显式连接不同名信号
);
```

## 7.6 端口匹配规则

```
┌─────────────────────────────────────────────────────────────┐
│                    端口匹配规则                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   规则1: 位宽匹配                                            │
│   ┌─────────────────────────────────────────┐              │
│   │  input  [7:0] data_in;  // 8位           │              │
│   │  logic [7:0] data;      // 8位 ✓ 匹配    │              │
│   │  logic [3:0] data;      // 4位 ✗ 警告    │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
│   规则2: 类型兼容                                            │
│   ┌─────────────────────────────────────────┐              │
│   │  input  logic   sig;   // logic端口      │              │
│   │  wire         sig;     // wire ✓ 兼容    │              │
│   │  real         sig;     // real ✗ 不兼容  │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
│   规则3: 方向匹配                                            │
│   ┌─────────────────────────────────────────┐              │
│   │  input  → 可连接 wire, reg, logic        │              │
│   │  output → 可连接 wire, reg, logic        │              │
│   │  inout  → 必须连接 wire                  │              │
│   └─────────────────────────────────────────┘              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 7.7 端口连接方式对比总结

| 方式 | 语法 | 优点 | 推荐度 |
|------|------|------|--------|
| **按名称** | `.port(sig)` | 清晰、安全 | ★★★★★ |
| **按位置** | `(sig1, sig2, ...)` | 简洁 | ★★☆☆☆ |
| **隐式 `.*`** | `(.*)` | 最简洁 | ★★★★☆ |
| **混合** | `(.*, .port(sig))` | 灵活 | ★★★★☆ |
| **接口** | `(interface.modport)` | 可重用 | ★★★★★ |

## 7.8 关键要点总结

| 要点 | 说明 |
|------|------|
| **按名称连接** | 推荐方式，清晰易维护 |
| **按位置连接** | 传统方式，易出错，不推荐 |
| **接口连接** | 使用modport，简化大型设计 |
| **隐式连接 `.*`** | 自动连接同名信号 |
| **位宽匹配** | 不匹配会产生警告或X态 |
| **未连接端口** | 输入为Z，输出被忽略 |
